CalistoLab


CalistoLab has been developed to replace the software usually used in the laboratories of physics and electronics, focusing on giving the student the ability to manipulate data in an interactive way.

Although the proyect is thought to be used by undergraduate students, secondary students can also benefit of its user-friendly interface as a first contact with a physics laboratory.

The user can import CSV files to work on previously recorded data or create a new blank chart to fill it while is making the measurements. This app gives the opportunity to both work in a classical way by editing the data on a chart or in a more interactive way by manipulating it directly on a graphical interface. Charts can be exported to CSV and to TEX, and plots can be saved as PNG or PDF.

Supported web navigators

Currently, CalistoLab works perfectly on Mozilla. Exporting to PDF doesn't work on Google Chrome unless PDF Viewer extension is installed. CalistoLab hasn't been tested on any other web browser.
